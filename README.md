## 安裝 Node.js
#### Windows  
下載地址: https://nodejs.org/en/  
#### Ubuntu  
>sudo apt-get update  
>sudo apt-get install nodejs  
>sudo apt install npm  
### 檢查是否安裝完成
>node -v

## 安裝vue  
>npm config set registry http://registry.npmjs.org/  
>npm install vue  

## 安裝vue-cli套件  
>npm install -g @vue/cli  
>npm install -g @vue/cli-init  

### 檢查Vue版本  
>vue -V  

## 建立專案(拉取模板)
>vue init webpack my-project  

**tip**

>cd my-project

>npm run dev
